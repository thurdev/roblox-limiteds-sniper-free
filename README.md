## Installation

```bash
# Clone this repository
$ git clone https://github.com/thurdev/roblox-limiteds-sniper-free.git

# Go into the repository
$ cd roblox-limiteds-sniper-free

# Install dependencies
$ yarn install

# Setup .env
$ cp .env.example .env # watch tutorial

# Run the app
$ yarn start
```
